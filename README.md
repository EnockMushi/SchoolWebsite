# School Management System (SMS)

A modern, responsive School Management System built with PHP, MySQL, and Bootstrap 5.

## 🚀 Features

- **Multi-Role Access**: Admin, Headmaster, Teacher, and Parent portals.
- **Dynamic Student Handbook**: Manage school policies directly from the dashboard.
- **School Calendar**: Automatically generated from announcements and events.
- **Modern UI**: Glassmorphism design with full Dark Mode support.
- **Security**: Role-based access control, PDO prepared statements, and secure authentication.

## 🛠️ Installation

1. Clone the repository to your local server (WAMP/XAMPP).
2. Create a database named `school_db`.
3. Import the `database/schema.sql` file into your MySQL database.
4. Update `sms/includes/db.php` with your database credentials.
5. Access the portal at `http://localhost/SCHOOLAPP/index.php`.
